# Game Data Cleaning with Pandas & Numpy

## Overview

This project demonstrates a complete data cleaning workflow using Python and Pandas on a video game dataset.

## Tasks Performed

- Loaded CSV dataset
- Inspected data types
- Checked missing values
- Removed duplicate records
- Dropped columns with excessive missing values
- Converted release dates to datetime
- Filled missing categorical values appropriately
- Removed rows with missing identifiers
- Exported the cleaned dataset

## Technologies
- Pandas
- Numpy

## Files

- `Game_info.csv` - Raw dataset
- `Cleaned_Game_info.csv` - Cleaned dataset
- `clean.py` - Data cleaning script
